# riff
this is realtime diff project

## Require
```
npm install electron-packager -g
npm install electron-builder -g
```


## Use
```
# Mac
clone https://github.com/richellin/electron-diff.git
npm install
npm run build:osx
open dist/osx/Riff-darwin-x64/Riff.app
```
